## Ekran Görüntüsü Alma İpuçları

### Windows
- **Tam Ekran**: `Win + PrtScn` (Screenshots klasörüne kaydeder)
- **Bölge Seçimi**: `Win + Shift + S` (Snipping Tool)

### macOS
- **Tam Ekran**: `Cmd + Shift + 3`
- **Bölge Seçimi**: `Cmd + Shift + 4`

### Linux
- **Tam Ekran**: `PrtScn`
- **Bölge Seçimi**: `Shift + PrtScn`

## Önerilen Araçlar

- **Postman**: API endpoint'lerini test etmek ve ekran görüntüsü almak için
- **Browser DevTools**: JSON yanıtlarını güzel formatlı görmek için
- **Snagit/Greenshot**: Profesyonel ekran görüntüleri için (opsiyonel)

## Kullanım Talimatları

1. **Sunum Öncesi**: Tüm ekran görüntülerini bu klasöre kaydedin
2. **Sunum Sırasında**: Demo başarısız olursa bu görüntüleri açın
3. **Yedekleme**: Ekran görüntülerini USB'ye de kopyalayın

## Not

- Ekran görüntüleri **yüksek çözünürlükte** olmalı (1920x1080 veya üzeri)
- JSON yanıtları **pretty-print** formatında olmalı (okunabilir)
- Hassas bilgi (API anahtarları, şifreler) **görünmemeli**
- Türkçe karakterler **düzgün görünmeli**
